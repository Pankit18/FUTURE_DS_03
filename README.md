# FUTURE_DS_03 – Task 3: College Event Feedback Sentiment Analysis

This repository contains my Task 3 submission for the **Future Interns – Data Science & Analytics** internship.

## Objective
Analyze student feedback for college events and classify sentiment as **Positive**, **Neutral**, or **Negative** using Python.

## Files
- `College_Event_Feedback.csv` – input dataset
- `Task3_Sentiment_Analysis.ipynb` – main Jupyter/Colab notebook
- `Task3_Sentiment_Analysis_Report.pdf` – summarized report
- `Screenshots/` – output graphs from the notebook

## Tools Used
- Python (Google Colab)
- Libraries: Pandas, Matplotlib, Seaborn, TextBlob

## Key Insights
- Most feedback is **Positive**.
- TechFest and Sports Meet received the highest ratings.
- Orientation received comparatively lower sentiment.
- The dataset indicates strong student satisfaction trends overall.
